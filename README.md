# 月面社会ビジュアルシミュレーター

`日本の国際宇宙探査シナリオ案2025` をもとに、月面社会の発展段階、社会機能、産業、依存関係、出典を直感的に理解するための React + TypeScript 製シミュレーターです。

## 目的

このアプリは、原典文書の要点を単に要約するのではなく、以下を操作しながら理解できる形に再構成することを目的にしています。

- 月面社会は、科学、生活、建設、輸送、電力、通信、測位、資源利用、データ、産業の相互依存で成立する
- 月面活動は、2020年代の準備から、2030年代のインフラ整備、2040年代の社会形成へ段階的に発展する
- 2040年代の40人規模常時滞在構想は、複数のインフラが揃って初めて成立する
- 構想案であり、確定計画ではないことを出典付きで理解できる

## 起動方法

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

## データ構造

シナリオデータは `src/data/` に JSON として分離しています。

- `eras.json`: 年代区分
- `zones.json`: 月面・月近傍のゾーン
- `actors.json`: 主体
- `assets.json`: 社会を構成する主要アセット
- `activities.json`: 活動
- `industries.json`: 産業候補
- `dependencies.json`: 依存関係
- `narrative.json`: ストーリーステップ
- `sourceIndex.json`: 出典索引
- `labels.ja.json`: UIラベル辞書
- `filterDefaults.json`: 初期フィルタ

型定義は `src/types/scenario.ts` にあります。

## 今後の拡張方法

- `src/data/` を差し替えることで、別シナリオ文書にも転用可能
- `labels.ja.json` と同構造の英語辞書を追加すれば、多言語化の起点になる
- PDFの引用要約をより厳密にする場合は、`sourceIndex.json` に図表番号や該当節IDを追加する
- 依存関係ビューは将来 `React Flow` に差し替えて大規模グラフに対応できる
- 出典ビューは PDF ページプレビューや図表サムネイル連携へ拡張できる
- CMS 化する場合は `src/lib/data.ts` を API クライアントに置き換える

## 出典データの入れ方

1. 原典の章・図表・表を確認する
2. 新しい要素に `sourcePages` を必ず付与する
3. ページだけでなく、出典の意味を `sourceIndex.json` に短く要約する
4. 原典にない断定は避け、説明文では `構想` `想定` `候補` などの表現を使う

## 実装メモ

- 状態管理は `Zustand`
- URL クエリで `view`, `era`, `selected`, `step` を共有可能
- UI ロジックとデータは分離
- 初期データは PDF の第3章、第5章、第6章、第7章、第9章、第12章の内容をもとに構成
